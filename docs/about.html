About LendMatch

LendMatch is a next-generation credit decisioning platform powered by advanced machine learning algorithms. Our mission is to provide transparent, fair, and instant loan assessments.

How It Works

  • Data-Driven: Trained on historical LendingClub accepted/rejected data (2007–2018Q4) when provided in data/raw.
  • Advanced AI: Uses gradient-boosting models to predict approval odds, default risk, and risk-based interest rates.
  • Fairness Focused: We carefully engineer features to avoid bias, focusing on financial indicators like DTI, FICO score, and employment history.

Technology Stack

Built with Python (FastAPI, Pandas, scikit-learn) and a modern responsive frontend.
